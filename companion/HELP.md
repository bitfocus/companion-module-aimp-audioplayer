# Setup
To use the module, you need to install a plugin (add-on) in the AIMP player itself. The plugin works with player version 5.40 and later. You can download the plugin from the add-ons catalog on the player's website or by following this link.

# Description

This module expands the player's capabilities for events. You can customize the buttons to play a specific track or playlist.

Track and playlist navigation is supported. Track and playlist information can be displayed. Command-based navigation is duplicated in the player's graphical interface, just as navigation actions on a PC are duplicated in the feedback information about the selected track and playlist.

There's a switch for automatically playing the next track, which allows you to disable automatic playback. This is useful when switching from background music playback at an event to the main event—specific playback of specific tracks.

Working with the player module allows you to partially replicate QLab's functionality for amateur use, even on Windows.
